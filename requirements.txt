# ==============================================================================
# Installation Instructions for Jetson Nano 2GB
# ==============================================================================
#
# 1. Update system packages:
#    sudo apt-get update && sudo apt-get upgrade
#
# 2. Install CUDA-enabled OpenCV (CRITICAL for performance):
#    sudo apt-get install python3-opencv
#    
#    This provides OpenCV with CUDA support, significantly faster than pip version.
#    Verify CUDA is enabled:
#    python3.9 -c "import cv2; print('CUDA devices:', cv2.cuda.getCudaEnabledDeviceCount())"
#
# 3. Install other system dependencies:
#    sudo apt-get install python3.9 python3-pip python3.9-dev
#    sudo apt-get install git-lfs
#    sudo apt-get install gstreamer1.0-tools gstreamer1.0-plugins-good \
#                         gstreamer1.0-plugins-bad gstreamer1.0-plugins-ugly \
#                         gstreamer1.0-libav
#
# 4. Install Python packages from requirements.txt:
#    pip3 install -r requirements.txt
#
#    OR for system-wide installation:
#    sudo pip3 install -r requirements.txt
#
#    Note: opencv-python is commented out in requirements.txt because
#    we're using the CUDA-enabled system version installed in step 2
#
# 5. Initialize Git LFS and pull model files:
#    git lfs pull
#
# 6. Enable jtop service (for hardware monitoring):
#    sudo systemctl restart jtop.service
#
# ==============================================================================
